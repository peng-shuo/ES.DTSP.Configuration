# review_agent 专用技能

## 最小流程（低 token）

1. 对照 `requirements/specs` 查偏差。
2. 按严重级别输出问题：高 -> 中 -> 低。
3. 每个问题仅给：证据、影响、建议。

## 输出模板（强制）

- 结论：发现的问题清单（无问题也要写“未发现问题”）。
- 风险：残余风险/测试缺口。
- 待决策：是否阻断合入、是否降级处理。

## 约束

1. 以问题为主，不做长总结。
2. 结论必须有证据路径。
