# demand_agent 专用技能

## 最小流程（低 token）

1. 只读：`requirements/roadmap/specs/adr` 和用户故事来源。
2. 只产出：`范围`、`AC`、`待决策` 三项。
3. 冲突优先报“范围冲突 > 术语冲突 > 阶段冲突”。

## 输出模板（强制）

- 结论：最多 3 条。
- 风险：最多 3 条。
- 待决策：最多 3 条。

## 约束

1. 不给实现细节代码。
2. 不绑定具体技术库。
3. 新增需求必须可验收。
